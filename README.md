  ## Getting Started
  
  To get a local copy up and running follow these simple steps.

  ### Prerequisites
  
npm

    ```
    
  ## Run the project locally
  
   Clone the repo
     ```sh
     git clone https://github.com/unnot1ced/portfolio.git
     ``` <br>
   Install NPM packages
     ```sh
     npm install
     ``` <br>
   Run development server
     ```sh
     npm run dev
